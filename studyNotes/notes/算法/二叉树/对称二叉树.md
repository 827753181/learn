# 题目

请实现一个函数，用来判断一颗二叉树是不是对称的。注意，如果一个二叉树同此二叉树的镜像是同样的，定义其为对称的。

# 思路

-   两个根结点相等
-   左子树的右节点和右子树的左节点相同。
-   右子树的左节点和左子树的右节点相同。

递归所有节点满足以上条件即二叉树对称。
