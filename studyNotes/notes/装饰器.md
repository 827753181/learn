# 类装饰器

```ts
@log("111")
class a {}
```

等于下面

```ts
log("111")(a);
```

# 类成员装饰器

类成员装饰器应用于类中的单个成员-无论它们是属性，方法，getter 或 setter。该装饰器被调用时需要有以下三个参数：

```ts
class a {
	@getUser("111")
	user: string;

	@getMethodUser("222")
	getMUser() {}
}
```

等于下面

```ts
class a {
	// target：要修改属性的对象。
	// name：要修改的属性的名称。
	// descriptor：要修改的属性描述符。从本质上讲，这是应该传递给Object.defineProperty的对象。
	user = getUser(target, name, descriptor);
	getMUser = getMethodUser(target, name, descriptor);
}
```
